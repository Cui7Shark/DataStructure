# 数据结构与常见排序算法
## 数据结构
1. 链表
2. 栈
3. 队列
4. 树
5. 图-- 可以使用邻接链表来存储稀疏图，使用二叉搜索树也许会更好，对于查找和插入删除来说。
## 排序算法 -- 稳定性 -- 平均时间复杂度
1. 快排 -- 不稳定 -- O(nlogn)
2. 冒泡 -- 稳定 -- O(n^2)
3. 归并排序 -- 稳定 -- O(nlogn)
4. 堆排序 -- 不稳定 -- O(nlogn)
